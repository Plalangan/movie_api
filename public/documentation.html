Documentation

This API is the go to resource when it comes to movie titles, actors, characters, and directors. Users will be able to sign up, update their personal information, and create a list of their favorite movies

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing a image. title, description, genre, and directors.

Example:

{ "Title": "Avengers: Endgame", "Description": "After the devastating events of Avengers: Infinity War (2018), the universe is in ruins. With the help of remaining allies, the Avengers assemble once more in order to reverse Thanos' actions and restore balance to the universe.", "Genre": {"Name":"Superhero Films", "Description": "A superhero film, superhero movie or superhero motion picture is a film that is focused on the actions of one or more superheroes: individuals who usually possess superhuman abilities relative to a normal person and are dedicated to protecting the public."} "Director": {"Name": "Anthony Russo", "Joe Russo" } }

Get data about a movie genre /movies/genres/[Name] GET None A JSON object holding data about a movie genre.

Example:

{ "Name": "Superhero Films", "Description": ""A superhero film, superhero movie or superhero motion picture is a film that is focused on the actions of one or more superheroes: individuals who usually possess superhuman abilities relative to a normal person and are dedicated to protecting the public." }

Get data about a director /movies/directors/[Name] GET None A JSON object holding data about a director.

Example:

{"Name": "Anthony Russo", "Bio": "Anthony Russo was born on February 3, 1970 in Cleveland, Ohio, USA as Anthony J. Russo. He is a producer and director, known for..."} {"Name:" "Joe Russo", "Bio": "Joe Russo was born on July 8, 1971 in Cleveland, Ohio, USA as Joseph Vincent Russo. He is a producer and director, known for..."}

Add new user accounts /users POST A JSON object holding data about the user to add, structured like:

{ "Username" : "user1234", "Password" : "password1234", "Email" : "example@hotmail.com", "Birthday" : "01-01-1991" }

A JSON object holding data about the user that was added, including an ID:

{ "FavoriteMovies": [], "_id": "01234", "Username": "user1234", "Password": "password1234", "Email": "example@hotmail.com", "Birthday": "01-01-1991" }

Update a user's information /users/[Username] PUT A JSON object holding data about the user to add

Example:

{ "Username" : "Newuser1234", "Password" : "Newpassword1234", "Email" : "example@gmail.com", "Birthday" : "01-01-1991" }

A text message indicating the user's information was updated
Add movies to user's list of favorites /users/[Username]/Movies/[MovieID] POST None A message indicating the movie was added to the user's list of favorites
Remove movies from user's list of favorites /users/[Username]/Movies/[MovieID] DELETE None A message indicating the movie was removed from the user's list of favorites
Remove user accounts /users/[Username] DELETE None A message indicating the user's account was deleted
